Material Design Focused

Keeping in mind elements and principles of Material Design, I have created a Dashboard that incorporates components and animations that provide more feedback to users. Everything just to provide a more unified user experience.

Easy to work with

I have provided necessary documentation and/or tutorial in order to make it easier for you to include it in your projects. By the way, I will provide support via Google+ or Hangouts.



FEATURES

License Checker Tool

PaperBoard will check if the app was actually installed from Google Play Store or if it's pirate. It will automatically disable the app funcionality if it detects app is pirate.

Changelog

PaperBoard will show a dialog with the changelog of your latest updates with every new version of your app.

Category-based Icons Grid

PaperBoard will separate your icons in Tabbed Categories and automatically show the icons in each one. Here, user will be able to see all the themed icons in your pack.

Cloud Wallpapers

PaperBoard includes support for Cloud Wallpapers. Users can download and apply them. This comes along with Muzei support.

Icon Request Tool

PaperBoard includes and provides a tool for you to allow users create Icon Requests and send them to you.

Apply Section

PaperBoard provides a section where user can apply the Icon Pack to 26 different launchers. It can detect if certain launcher is installed or not.

Credits Section

PaperBoard includes a section where you provide credits to the app developers (including you) and the implemented libraries.

Animations

PaperBoard has been developed with many cool but subtle animations to make your app look even better.

And Even More

PaperBoard still have a lot of features to provide and offer. I will continue working and learning in order to make it the best one.
